DESPLIEGUE WEB

Este paquete ya está preparado para publicarlo como página web y abrirlo mediante enlace.

Opciones sencillas:
1. GitHub Pages
2. Netlify
3. Vercel
4. Cualquier hosting estático

Pasos básicos:
- Sube todos los archivos tal como están.
- Publica la carpeta raíz.
- Obtendrás un enlace web.
- Ese enlace se abrirá en iPhone, Android, iPad y ordenador.

APP EN MÓVIL
- Android: el navegador puede mostrar "Instalar".
- iPhone: abre el enlace en Safari > Compartir > Añadir a pantalla de inicio.

ACTUALIZACIÓN iPHONE / SONIDOS
- El campo de respuesta ahora usa:
  type="text" + inputmode="numeric" + pattern="[0-9]*" + enterkeyhint="done"
- En muchos iPhone esto muestra una tecla de confirmación más amigable que con type="number".
- Aun así, Safari/iOS no siempre garantiza un botón Enter visible en todos los teclados numéricos.
- Por eso se mantiene también el botón "Aceptar" de la interfaz.

SONIDOS
- Se han añadido sonidos generados por Web Audio:
  - acierto
  - fallo
- No necesitan archivos mp3 y funcionan bien en móvil y escritorio tras la primera interacción del usuario.


PANEL NUMÉRICO V3
- Se ha añadido un panel numérico integrado en la propia web.
- Esto evita depender del botón Enter del teclado de iPhone.
- Se puede seguir usando el teclado físico en ordenador.
- El panel incluye:
  - números del 0 al 9
  - botón C para borrar
  - botón OK para enviar
- También incorpora un pequeño sonido táctil al pulsar botones.
